# Retail Store Sales - Week 2 Data Science Assignment

## Objective
Data Cleaning, EDA, and Sales Prediction using Machine Learning.

## Dataset
- Source: Kaggle (Retail Store Sales: Dirty for Data Cleaning)
- 12,575 rows, 11 columns

## Tasks Completed
- Task 1: Advanced Data Cleaning (missing values, outliers, encoding)
- Task 2: Exploratory Data Analysis (5+ charts, insights)
- Task 3: Sales Prediction Model (Linear Regression + Random Forest)

## Model Results
| Model | RMSE | MAE | R2 Score |
|-------|------|-----|----------|
| Linear Regression | 35.44 | 25.58 | 0.84 |
| Random Forest | 23.40 | 7.85 | 0.93 |

## Best Model: Random Forest (93% accuracy!)
